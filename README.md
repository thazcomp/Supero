# Supero Test
## Repo Reader

Aplicativo teste para a empresa Supero, nomeado Repo Reader, responsável por ler os Repositórios Públicos do GitHub que utilizam a linguagem Java, exibir para o usuário e proporcionar a opção de ver a lista de Pulls e detalhes de cada Repositório.
Fora utilizada a linguagem Kotlin para realização de tal, assim como o padrão de projeto MVVM e as seguintes bibliotecas de terceiros:

* Koin (injeção de dependecias)
* Retrofit (comunicação com a API)
* RX
* Glide

Entre outras não tão impactantes.
O Desenvolvimento do Projeto **NÃO** foi focado em layout e detalhes gráficos, e sim, em construir um código limpo, enxuto e de fácil entendimento, reduzindo a necessidade de comentários e explicações.
Apenas algumas informações como exemplos são mostradadas nas telas, sendo que o aplicativo não está maior pelo curto prazo de entrega.
